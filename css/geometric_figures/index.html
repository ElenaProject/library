Геометрические фигуры

1.

Для начала нeобходимо понять как сделать треугольники - из их комбинации составляются более сложные фигуры.

Они формируются из блоков, имеющих width: 0; height: 0; а точнее из их рамок. Из рамки каждой стороны мы получаем отдельный треугольник.
